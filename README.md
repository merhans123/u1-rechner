# U1-Rechner – Entscheidungshilfe für Arbeitgeber

**Version:** v1.0.1  
**Status:** stabil

## Funktionen
- Wirtschaftlichkeitsvergleich der U1-Tarife
- Umschaltbar: 30-igstel-Methode oder Arbeitstage
- Arbeitstage: 3 / 4 / 5-Tage-Woche
- Break-Even-Tag je Tarif
- Hinweise für nie optimale Tarife

## Datenschutz
- Keine personenbezogenen Daten
- Keine Speicherung
- Lokale Berechnung im Browser

## Nutzung
```bash
python3 -m http.server 8000
```
